# 🧮 Basic JavaScript Calculator

A simple calculator built using JavaScript that performs basic arithmetic operations and updates results dynamically 
on the webpage.

## 🚀 Features
- Addition, subtraction, multiplication, and division
- Real-time result display
- Interactive UI using DOM manipulation
- Beginner-friendly and easy to understand

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 📚 What I Learned
- How to use JavaScript functions
- Working with variables and operators
- DOM manipulation (`getElementById`, `textContent`)
- Handling user interactions

## 💡 How It Works
The calculator takes input values, performs the selected operation using JavaScript functions, and displays 
the result instantly on the page.
